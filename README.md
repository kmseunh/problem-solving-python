# Algorithm

올바른 알고리즘: 어떠한 경우에도 실행 결과가 똑같이 나오는 것.
<br>

### 목표

- 문제 해결 능력 강화

- 코드 효율성 개선

- 문제 해결 전략 습득
<br>

### 규칙

- 매일 최소 1개 이상의 알고리즘 문제를 풀고 어려운 문제는 풀이과정을 커밋합니다.
- 최대 1시간 정도로 시간제한을 두고 문제를 풉니다.
- 시간내에 풀지 못하면 풀이를 보고, 정답코드에서 몰랐던 부분을 공부합니다.
- 풀지 못한 문제는 꾸준히 반복해서 풀어봅니다.
